faqbuster

Add annotations as iframe tooltips.

To run a http://is.gd/shtikipedia clone on localhost:

  1. Clone https://github.com/thedod/faqbuster and cd there.
  2. Drag this bookmarklet to the toolbar: fqb (Note that it loads the the external js and css files).
  3. Run python -m SimpleHTTPServer;SimpleHTTPServer (will serve current folder on port 8080)
  4. Go to the original gov page and click on the bookmarklet.

This repository is for people who want to contribute CSS/JS to the site independent part of the project.

To contribute content to shtikipedia, please go here.
